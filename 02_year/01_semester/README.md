## 🗓️ 1st Semester Modules

### 🧮 Algorísmia (Algorithm Design I)
**Objective:** Analyze mathematical boundaries of code efficiency, establishing rigorous runtime upper and lower bounds for processing constraints.
* **Key Challenges:** Formulating explicit inductive complexity recurrences and choosing between greedy or divide-and-conquer strategy branches.
* **Approach:** Execution of runtime comparisons using Big-O notation proofs, implementing recursive divide-and-conquer frameworks, and evaluating greedy optimization heuristics.
* **Tech Stack:** Java / C++.

---

### 🧠 EC2 (Computer Architecture II)
**Objective:** Implement and optimize performance execution loops directly inside custom microprocessing computing hardware layouts.
* **Key Challenges:** Resolving structural data conflicts and optimizing branching delay penalties inside parallel instruction executions.
* **Approach:** Designing pipeline structural controls, managing CPU hazards, calculating memory speeds, and structuring optimized cache data line configurations.
* **Tech Stack:** Assembly Language, Hardware Simulators.

---

### 🐚 SO (Operating Systems I)
**Objective:** Master kernel-level execution boundaries, task scheduling policies, and multi-program resource distribution mechanics.
* **Key Challenges:** Preventing execution race conditions, resolving thread deadlocks, and mapping multi-tier virtual addresses.
* **Approach:** Scripting safe multi-threaded concurrency controls, configuring mutual exclusion locks (Mutex / Semaphores), and tracing physical data lookups inside virtual memory paging frames.
* **Tech Stack:** C (Linux POSIX Threads).

---

### 🌐 Xarxes (Computer Networks)
**Objective:** Analyze global internet telematic infrastructures across structural protocol layers, ensuring robust data exchange pipelines.
* **Key Challenges:** Debugging packet drop limits and managing raw stream serialization architectures.
* **Approach:** Tracing data flows down the conceptual OSI layer model, implementing network socket connections, handling asynchronous TCP/IP streams, and decoding packet framing patterns.
* **Tech Stack:** C / Python (Socket API), Wireshark.
