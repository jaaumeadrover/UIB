# 🎓 BSc Computer Science — Year 2 

This repository directory organizes the complete coursework, laboratory code architectures, and technical project implementations completed during my second year within the Computer Science undergraduate curriculum at UIB. 

Year 2 marks the explicit transition from elementary programming into core systems engineering, covering algorithmic bounds, bare-metal hardware performance, network topologies, and transactional data management.
