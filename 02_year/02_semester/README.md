## 🗓️ 2nd Semester Modules

### 🎚️ ACSII (Advanced Computer Architecture)
**Objective:** Exploit fine-grained hardware instruction execution streams to scale throughput speeds across modern multi-core processing architectures.
* **Key Challenges:** Eliminating thread dependencies and ensuring data consistency across parallel hardware cache structures.
* **Approach:** Evaluation of superscalar out-of-order schedulers, tracking vector processing registers, and mapping multi-core directory cache coherency loops.
* **Tech Stack:** Hardware Simulators, VHDL.

---

### ◀️ Algorísmia II (Algorithm Design II)
**Objective:** Translate network, pathing, and dependency properties into advanced graph data structures to compute optimized topological results.
* **Key Challenges:** Formulating multi-constraint network path models and analyzing the boundaries of mathematically intractable (NP-complete) spaces.
* **Approach:** Designing advanced tree tracking metrics, implementing graph searches (BFS / DFS), executing shortest-path logic (Dijkstra / Bellman-Ford), and calculating Maximum Flow bottlenecks.
* **Tech Stack:** Java / C++.

---

### 🗄️ Base de Dades (Database Systems I)
**Objective:** Model, structure, and query robust relational data storage architectures under strict relational integrity constraints.
* **Key Challenges:** Mapping complex multi-table relationships while eliminating dangerous entry redundancies through structural normalization.
* **Approach:** Transforming real-world rules into logical Entity-Relationship (E-R) schemas, executing 3NF / BCNF normalizations, and scripting high-performance indexed SQL queries.
* **Tech Stack:** MySQL / PostgreSQL.

---

### 🏗️ Enginyeria del Software (Software Engineering)
**Objective:** Apply robust, industry-standard engineering methodologies and OOP design choices to build scalable, production-grade software platforms.
* **Key Challenges:** Documenting dynamic, non-conflicting structural component charts and coordinating decoupled team developer cycles.
* **Approach:** Mapping software plans via Unified Modeling Language (UML) class/sequence matrices, executing Agile Scrum sprint milestones, and deploying automated unit testing layers.
* **Tech Stack:** Java / Git.

---

### 💻 Sistemes Operatius II (Operating Systems II)
**Objective:** Scale low-level system execution frameworks across multi-node, non-monolithic computing networks and shared configurations.
* **Key Challenges:** Managing resource locking speeds across distributed nodes and handling high-concurrency non-blocking input/output channels.
* **Approach:** Configuring distributed directory networks, optimizing non-blocking asynchronous event loops, and tracking transaction storage inside virtual file system tables.
* **Tech Stack:** C / Linux Bash.

---

### ⚙️ Teoria de la Computació (Automata Theory)
**Objective:** Define the abstract mathematical capabilities, syntax parsing thresholds, and uncomputable limitations regulating digital machines.
* **Key Challenges:** Proving structural language classifications and constructing formal mathematical state transition deciders.
* **Approach:** Constructing Deterministic and Non-Deterministic Finite Automata (DFA / NFA), processing regular Expressions, defining Context-Free Grammars, and executing abstract Turing Machine configurations.
* **Tech Stack:** JFlex, JCUP.
