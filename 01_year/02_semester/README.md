## 🗓️ 2nd Semester Modules

### 📐 Linear Algebra
**Objective:** Provide matrix and tensor vector space foundations to support computer graphics, multidimensional feature transformations, and matrix operations.
* **Key Challenges:** Visualizing high-dimensional spatial transformations and calculating eigenvalue dependencies.
* **Approach:** Application of Gaussian elimination, vector subspaces, orthogonal bases, matrix factorization, and diagonalizations.
* **Tech Stack:** MATLAB.

---

### 📊 Statistics
**Objective:** Apply probability distributions and statistical inference mechanisms to process data samples, validate hypotheses, and model errors.
* **Key Challenges:** Discerning appropriate parametric models for highly asymmetrical real-world distribution data.
* **Approach:** Constructing regression frameworks, determining confidence intervals, calculating ANOVA variances, and deploying data validation boundaries.
* **Tech Stack:** R / RStudio.

---

### 🔌 Computer Structure (Estructura de Computadors)
**Objective:** Analyze the low-level physical interfaces of computational units, mapping software binaries to CPU registers, memory hierarchies, and system buses.
* **Key Challenges:** Deconstructing manual memory segmentation blocks and debugging register constraints.
* **Approach:** Programming direct low-level micro-operations using raw **Assembly Language**, parsing bus synchronization cycles, evaluating cache hits/misses, and assessing hardware interrupt mechanics.
* **Tech Stack:** Assembly (RISC-V), Venus Simulator / Assembler.

---

### ⚡ Physics
**Objective:** Analyze the thermodynamic and electromagnetic laws dictating the structural physics of physical hardware and electronic computational systems.
* **Key Challenges:** Translating field differential formulas into discretized physical hardware models.
* **Approach:** Rigorous modeling of circuit laws, electric potentials, magnetic flux interactions, and semiconductor diode behavior.
* **Tech Stack:** MATLAB.

---

### 🏗️ Coding II
**Objective:** Transition from foundational programming to full Object-Oriented Programming (OOP) paradigms, implementing efficient structural memory layouts.
* **Key Challenges:** Managing class inheritances, polymorphism bounds, explicit encapsulation protocols, and complex sorting data networks.
* **Approach:** Engineering clean modular projects utilizing core encapsulation design patterns, polymorphism, abstract typing, error handling blocks, and basic data structures.
* **Tech Stack:** Java (NetBeans IDE).
