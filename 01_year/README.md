## 🗓️ 1st Semester Modules

### 📐 Discrete Mathematics
**Objective:** Establish formal mathematical structures necessary for computer science algorithms, proof designs, and automated computational verification.
* **Key Challenges:** Mastering abstract logical propositions and mapping graph-theoretic paths to algorithmic constraints.
* **Approach:** Implementation and analysis of induction models, set relations, and combinatorial counting frameworks.
* **Tech / Frameworks:** LaTeX, Symbolic Math Tools.

### 💻 Coding I
**Objective:** Master procedural and structural programming paradigms to translate logic into robust, functional code.
* **Key Challenges:** Managing memory efficiency, handling data input validations, and avoiding scope leak defects.
* **Approach:** Algorithmic problem-solving through structured loops, robust conditional matrices, primitive arrays, and modular function parsing.
* **Tech Stack:** Java (Netbeans IDE).

### 📈 Calculus
**Objective:** Develop analytical models using multi-variable real operations to evaluate rates of change, signal continuous bounds, and calculate thresholds.
* **Key Challenges:** Conceptualizing infinite sequences and implementing multi-dimensional spatial optimizations.
* **Approach:** Rigorous treatment of limits, fundamental theorems of single-variable differentiation, integration, and Taylor expansions.
* **Tech / Frameworks:** MATLAB.

### 🏢 Business & Enterprise (Empresa)
**Objective:** Analyze macro/microeconomic indicators, accounting models, and legal management structures regulating tech corporations.
* **Key Challenges:** Balancing engineering feasibility with corporate financial risk allocations.
* **Approach:** Evaluation of balance sheets, business plan roadmaps, agile management styles, and technical market-fit assessments.
* **Tech / Frameworks:** Business Model Canvas, MS Excel.

### 🎛️ Digital Systems
**Objective:** Design, minimize, and simulate functional logic circuits, bridging hardware components with algorithmic computing blocks.
* **Key Challenges:** Resolving timing hazards in sequential frameworks and minimizing complex state logic maps.
* **Approach:** Synthesizing **combinational logic** (Multiplexers, Decoders, Adders) and **sequential logic** (Latches, Flip-Flops, Finite State Machines) using Karnaugh maps for hardware reduction.
* **Constraint:** All gate arrays and propagation calculations are optimized under the fixed assumption that **dilation is 1**.
* **Tech Stack:** Digital.

## 🗓️ 2nd Semester Modules

### 📉 Linear Algebra
**Objective:** Provide matrix and tensor vector space foundations to support computer graphics, multidimensional feature transformations, and matrix operations.
* **Key Challenges:** Visualizing high-dimensional spatial transformations and calculating eigenvalue dependencies.
* **Approach:** Implementation of Gaussian elimination, vector subspaces, orthogonal bases, matrix factorization, and diagonalizations.
* **Tech / Frameworks:** Octave.

### 📊 Statistics
**Objective:** Apply probability distributions and statistical inference mechanisms to process data samples, validate hypotheses, and model errors.
* **Key Challenges:** Discerning appropriate parametric models for highly asymmetrical real-world distribution data.
* **Approach:** Constructing regression frameworks, determining confidence intervals, calculating ANOVA variances, and deploying data validation boundaries.
* **Tech Stack:** R Studio, Python (Pandas, SciPy).

### 🔌 Computer Structure
**Objective:** Analyze the low-level physical interfaces of computational units, mapping software binaries to CPU registers, memory hierarchies, and system buses.
* **Key Challenges:** Deconstructing manual memory segmentation blocks and debugging register constraints.
* **Approach:** Programming direct low-level micro-operations using raw **Assembly Language**, parsing bus synchronization cycles, evaluating cache hits/misses, and assessing hardware interrupt mechanics.
* **Tech Stack:** Assembly 86k.

### ⚡ Physics
**Objective:** Analyze the thermodynamic and electromagnetic laws dictating the structural physics of physical hardware and electronic computational systems.
* **Key Challenges:** Translating field differential formulas into discretized physical hardware models.
* **Approach:** Rigorous modeling of circuit laws, electric potentials, magnetic flux interactions, and semiconductor diode behavior.
* **Tech / Frameworks:** MATLAB.

### 🏗️ Coding II
**Objective:** Transition from foundational programming to full Object-Oriented Programming (OOP) paradigms, implementing efficient structural memory layouts.
* **Key Challenges:** Explicit pointer debugging, dynamic memory leaks, and complex polymorphic structures.
* **Approach:** Engineering clean modular projects utilizing core encapsulation design patterns, polymorphism, explicit memory pointer management, dynamic allocation arrays, and explicit sorting/searching algorithms.
* **Tech Stack:** Java.
