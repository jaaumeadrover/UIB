# 🎓 BSc Computer Science — Year 3

This repository directory organizes the complete coursework, laboratory code architectures, and technical project implementations completed during my third year within the Computer Science undergraduate curriculum at UIB.

Year 3 represents the core specialization phase of the degree, bridging advanced theoretical limits with production-grade backend engineering. Key milestones include implementing full language compilers, managing multi-core thread concurrency, constructing real-time 3D graphics rendering pipelines, and deploying heuristic search agents.
