## 🗓️ 2nd Semester Modules

### 📈 Algorismes Avançats (Advanced Algorithms)
**Objective:** Formulate specialized algorithm architectures to compute mathematically complex, multi-constraint, or NP-hard problems within optimized execution limits.
* **Key Challenges:** Mapping physical layout constraints to abstract network flow models and proving approximation optimization margins.
* **Approach:** Deploying complex Dynamic Programming tables, calculating Network Flows (Max-Flow/Min-Cut bottlenecks), evaluating randomized heuristics, and managing approximation algorithms for NP-complete limits.
* **Tech Stack:** Python / C++.

---

### 📝 English for Computer Science
**Objective:** Standardize professional technical communication, documentation delivery, and presentation frameworks used within international engineering environments.
* **Key Challenges:** Translating dense algorithmic structures into clean, unambiguous technical prose for diverse target stakeholder groups.
* **Approach:** Drafting standard system specification sheets, writing deployment logs, analyzing technical research publications, and refining professional project pitches.
* **Tech Stack:** LaTeX, Markdown.

---

### 🎨 Gràfica (Computer Graphics)
**Objective:** Construct real-time mathematical rendering environments to display virtual three-dimensional objects onto responsive physical frames.
* **Key Challenges:** Visualizing matrix transformations across moving frame spaces and writing low-latency GPU instructions.
* **Approach:** Building a full 3D graphics rendering pipeline, executing linear affine transformations (translation, rotation, perspective matrices), calculating reflection lighting models, and writing custom GPU rasterizer shaders.
* **Tech Stack:** C++ / C, OpenGL / WebGL, GLSL Shaders.

---

### 📜 Llenguatges (Programming Languages Theory)
**Objective:** Evaluate the formal mathematical frameworks, type constraints, and core semantics that define different programming language paradigms.
* **Key Challenges:** Shifting logical models away from standard imperative step-by-step code execution into declarative constraints.
* **Approach:** Tracing data flows through Functional and Logic paradigms, testing static vs. dynamic type safety frameworks, analyzing evaluation strategies (lazy vs. eager), and defining formal operational semantics.
* **Tech Stack:** Haskell, Prolog, Lisp.

---

### 🎚️ Multimèdia (Multimedia Systems)
**Objective:** develop a responsive, elegant website.
* **Key Challenges:** Responsive design, optimal load time etc..
* **Approach:** Using SVG images and suited formats.
* **Tech Stack:** HTML, CSS.
