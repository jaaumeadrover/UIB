## 🗓️ 1st Semester Modules

### 🌐 ADIU (Distributed Applications & User Interfaces)
**Objective:** Architect decoupled, scalable distributed computing clusters supporting responsive, real-time asynchronous multi-user interface flows.
* **Key Challenges:** Managing atomic data synchronizations across shifting network states and preventing UI rendering threads from blocking during heavy I/O operations.
* **Approach:** Separating backend business logic into distributed microservices, implementing secure API payloads, managing asynchronous network event loops, and designing reactive frontend state systems.
* **Tech Stack:** JavaScript / TypeScript, Node.js, React / Vue, REST APIs, WebSockets.

---

### 🧠 Aprenentatge Automàtic (Machine Learning)
**Objective:** Engineer robust mathematical models capable of parsing raw multidimensional data arrays to uncover latent structures and output automated predictive classifications.
* **Key Challenges:** Preventing model overfitting configurations and stabilizing convergence behavior across asymmetric feature scales.
* **Approach:** Executing rigorous feature extraction and normalization matrices, training supervised ensembles (Random Forests, Gradient Boosting Machines), evaluating deep neural configurations, and plotting cross-validation bounds.
* **Tech Stack:** Python, Scikit-Learn, PyTorch, Pandas, NumPy.

---

### ⛏️ Data Mining
**Objective:** Deploy analytical discovery frameworks over massive relational and unstructured datasets to identify patterns, anomalies, and descriptive trends.
* **Key Challenges:** Scaling algorithmic executions over dense data volumes and maintaining precision limits during heavy dimensional reductions.
* **Approach:** Designing frequent itemset pattern extractions (Apriori / FP-Growth), deploying unsupervised clustering groupings (K-Means, DBSCAN), and mapping Principal Component Analysis (PCA) feature spaces.
* **Tech Stack:** Python (SciPy, Scikit-Learn), RStudio.

---

### 🔮 Sistemes Intel·ligents (SI) (Intelligent Systems)
**Objective:** Formulate expert systems and autonomous software agents capable of simulating contextual reasoning loops under high operational uncertainty.
* **Key Challenges:** Encoding raw qualitative human expertise into rigid operational matrices and processing non-binary, shifting truth values.
* **Approach:** Designing formal structural semantic networks and knowledge graphs, writing targeted forward/backward chaining inference loops, and implementing mathematical **Fuzzy Logic** reasoning sets.
* **Tech Stack:** Python, Prolog, Fuzzy Logic Simulation Toolkits.

## 🗓️ 2nd Semester Modules

### 🏢 Practicullum (Professional Internship)
**Objective:** Integrate inside an enterprise technical engineering department to deploy production code, adapt to established software patterns, and operate within commercial development constraints.
* **Key Challenges:** Aligning personal development speeds with complex, legacy enterprise codebases and cross-functional team synchronizations.
* **Approach:** Contributing directly to real-world deployment sprints, handling system maintenance logs, writing automated testing parameters, and participating in corporate Agile/Scrum structures.
* **Tech Stack:** Industry-specific tech stack defined by the host organization enterprise environment.

---

### 🎓 Treball de Final de Grau (TFG — Final Degree Thesis)
**Objective:** Design, evaluate, and build an end-to-end Computer Vision **3D Reconstruction Pipeline** optimized for clinical **Diabetic Foot** diagnostic scanning and spatial morphological assessment.
* **Key Challenges:** Handling spatial sensor noise artifact layers, performing point cloud registrations across unaligned dynamic camera sweeps, and ensuring high topological fidelity for medical surface analysis.
* **Approach:** Capturing and processing multi-view **RGB-D datasets** recorded via an **Intel RealSense** depth camera ecosystem. Applying point cloud filtering primitives, computing iterative spatial alignment transforms (ICP registration matrices), and generating dense volumetric surface meshes for geometry processing.
* **Tech Stack:** Python / C++, Open3D, PCL (Point Cloud Library), Intel RealSense SDK, NumPy, Open3D-ML.
